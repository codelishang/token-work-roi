# 元衡品牌说明 / Yuanheng Brand Notes

项目中文名：**元衡**  
英文名：**Token Work ROI**

“元”表示词元、成本和原始记录。“衡”表示衡量、校准和取舍。这个名字想表达的是：把 AI 编程里的用量、费用、产出和模型策略放到同一套口径里看。

`元` points to tokens, cost, and original records. `衡` means measuring, calibrating, and weighing tradeoffs.

## Logo 含义 / Logo Meaning

图形标识由项目内自行绘制，未主动使用第三方品牌标志或素材模板。

The mark is drawn within this project and does not intentionally use third-party brand marks or stock templates.

图形由三部分组成：

- 中间竖线：表示统一口径。
- 左右弧线：表示衡量和校准。
- 彩色横线：表示把用量、成本和产出放在一起判断。

The mark has three parts:

- Center line: one shared measurement standard.
- Side curves: measuring and calibration.
- Color line: usage, cost, and output reviewed together.

颜色偏深，是为了适配工具栏、桌面图标、README 和报告封面。整体方向是清楚、稳定、不过度装饰。

The dark base keeps it usable in toolbars, desktop icons, README pages, and report covers. The goal is clear and stable, not decorative.

## 使用规则 / Usage

建议：

- 保持原始比例。
- 保留足够留白。
- 在浅色背景使用标准版本。
- 小尺寸场景优先使用 `token-work-icon.svg` 或 `token-work-icon.png`。

不建议：

- 叠加额外字母缩写。
- 加阴影、立体高光或图表装饰。
- 拉伸变形。
- 和第三方品牌图形组合成一个新图标。
